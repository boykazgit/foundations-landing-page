My First Landing Page 

This is a project as part of The Odin Project Full Stack development course.
We are supposed to make a landing page based on an already designed landing page.
The purpose of the project is the mainly practice using flexboxes and also to practice the basics I have learned up to this point.
It also encouraged me to do research of some css features which wasn't discussed in the course but was needed for the course.

I started the project with high hopes cause it would very wonderful if I were to create something complex as this.
There were few bottlenecks in creating this but overcame them and was able to produce this.

I also have high hopes going forward to learn more and to get better overall in webdevelopment.

Thanks to the odin community for providing such a wonderful resource for learning web development.
